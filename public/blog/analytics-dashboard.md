# Understanding Your Analytics Dashboard

Make data-driven decisions with DainoStore's built-in analytics. Track sales, understand customers, and identify growth opportunities.

---

## Overview

Your analytics dashboard provides:
- Real-time sales data
- Customer behavior insights
- Product performance metrics
- Marketing effectiveness tracking
- Trend analysis and forecasting

---

## Dashboard Overview

### Accessing Analytics

1. Go to **Analytics** in the admin sidebar
2. View the main dashboard
3. Use date picker to select time range
4. Compare to previous periods

## Setting Up Goals

### Conversion Goals

Track key actions:

1. Go to **Analytics > Goals**
2. Click **Create Goal**
3. Select goal type:
   - Purchase completed
   - Email signup
   - Account created
   - Page viewed

### Goal Funnels

Track multi-step processes:

**Checkout Funnel**:
```
Step 1: Cart (100%)
Step 2: Checkout (65%)
Step 3: Shipping (80%)
Step 4: Payment (90%)
Step 5: Complete (85%)
Overall: 40%
```

Identify where customers drop off.

## Common Questions

**Q: How often does data update?**
A: Sales data updates in real-time. Some aggregated metrics update hourly.

**Q: Can I see data from before I started?**
A: Only data since connecting DainoStore is available.

**Q: How do I track specific campaigns?**
A: Use UTM parameters in your links and view by source/campaign.

**Q: What affects my conversion rate?**
A: Site speed, product pages, checkout flow, pricing, trust signals.

## Analytics Best Practices

### Do

1. **Check daily** - Quick review of key metrics
2. **Compare periods** - Always use comparisons
3. **Segment data** - Break down by dimensions
4. **Act on insights** - Data should drive decisions
5. **Track trends** - Long-term patterns matter

### Don't

1. **Obsess over daily swings** - Look at trends
2. **Ignore context** - Seasonality, promotions affect data
3. **Vanity metrics** - Focus on actionable numbers
4. **Analysis paralysis** - Make decisions with good-enough data
