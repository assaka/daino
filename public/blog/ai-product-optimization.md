## AI-Powered Descriptions

### How It Works

1. You provide basic product information
2. AI analyzes the product category and attributes
3. Generates professional, converting copy
4. You review and edit as needed

### Using AI Description Generator

#### Step 1: Open Product

1. Go to **Catalog > Products**
2. Select a product
3. Click **AI Optimize** button

#### Step 2: Configure Options

| Option | Purpose |
|--------|---------|
| Tone | Professional, Casual, Luxury |
| Length | Short, Medium, Detailed |
| Focus | Features, Benefits, Technical |
| Include | Keywords, Specifications |

#### Step 3: Generate and Review

1. Click **Generate Description**
2. Review the AI output
3. Edit any specifics
4. Save changes

### Example Results

**Input**:
```
Product: Blue Cotton T-Shirt
Category: Men's Clothing
Material: 100% Organic Cotton
Size: S-XXL
```

**AI Output**:
```
Experience everyday comfort with our Blue Cotton T-Shirt,
crafted from 100% organic cotton for a soft, breathable feel.

Perfect for casual days or layering, this versatile tee
features a classic crew neck and relaxed fit that flatters
every body type.

Key Features:
- Premium organic cotton construction
- Breathable fabric ideal for all seasons
- Pre-shrunk for lasting fit
- Available in sizes S-XXL

Whether you're running errands or relaxing at home, this
essential tee delivers comfort without compromising style.
```

## AI Title Optimization

### Why Titles Matter

Your product title affects:
- Search rankings (SEO)
- Click-through rates
- Marketplace visibility
- Customer understanding

### Title Optimization Process

1. Go to **Catalog > Products**
2. Select products to optimize
3. Click **Bulk Actions > AI Optimize Titles**
4. Review suggestions
5. Apply selected changes

### Title Formula

AI generates titles following proven formulas:

**Standard format**:
```
[Brand] + [Product Type] + [Key Feature] + [Size/Variant]
```

**Example**:
```
Nike Air Max Running Shoes - Breathable Mesh - Men's Size 10
```

**E-commerce format**:
```
[Product Type] + [Material/Feature] + [Benefit] + [Specs]
```

**Example**:
```
Wireless Bluetooth Headphones | Noise Cancelling | 40H Battery | Black
```

## Marketplace Optimization

### Amazon Listings

AI creates Amazon-optimized content:

**Title** (max 200 chars):
- Front-loaded keywords
- Brand at start
- Key features included

**Bullet Points**:
- Benefit-focused
- Keyword-rich
- Scannable format

**Description**:
- HTML formatting
- Feature highlights
- Call to action

### eBay Listings

AI generates eBay-specific content:

**Title** (max 80 chars):
- Most important keywords first
- No special characters
- Clear product identification

**Description**:
- Professional HTML layout
- Feature sections
- Shipping and returns info

### Using Marketplace Optimization

1. Go to **Import & Export > Marketplace Hub**
2. Select marketplace (Amazon/eBay)
3. Enable **AI Optimization**
4. Select products
5. Click **Create Listings**

## SEO Meta Generation

### Automatic Meta Tags

AI generates:
- Meta titles
- Meta descriptions
- URL slugs
- Alt text for images

### Meta Title Optimization

**Formula**:
```
[Product Name] | [Key Benefit] | [Brand Name]
```

**Character limit**: 60 characters for best display

**Example**:
```
Organic Cotton T-Shirt | Soft & Breathable | EcoWear
```

### Meta Description Optimization

**Formula**:
```
[What it is] + [Key benefit] + [Call to action]
```

**Character limit**: 155 characters

**Example**:
```
Discover our premium organic cotton t-shirt. Ultra-soft,
breathable, and eco-friendly. Perfect for everyday comfort.
Shop now and feel the difference.
```

## Troubleshooting

### Poor Quality Output

**Solutions**:
- Provide more input data
- Choose appropriate settings
- Check product category
- Try regenerating

### Content Too Similar

**Solutions**:
- Vary input details
- Adjust temperature setting
- Edit uniquely per product
- Add product-specific info

### Translation Issues

**Solutions**:
- Review with native speaker
- Use simpler source text
- Check for idioms/slang
- Manual adjustment needed
